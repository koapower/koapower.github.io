Overview

We collaborated with a Taiwanese adult video company: "Model Media", and delivered the game as their first step towards the game industry.

The communication between two companies from different industries was a great experience to learn from a different perspective.

Technologies Used

  • Unity - Game engine for development
  • C# - Primary programming language
  • FFmpeg - Tool to deal with hundreds of video assets
  • InnoSetup - Tool to pack the game

Key Features

Story tree

The main story follows a tree structure to progress.

Massive video processing streamline

The game includes a huge amount of videos to process from our partner.

Development Process

Despite several major design changes, I spent considerable time communicating with our partner's executive for feasible solutions.

The project was developed before Gen AI came out. I designed and developed several tools from scratch to fulfill the need to process the significant amount of video assets from our partner. Also for debugging purposes.

Challenges & Solutions

Challenge 1: WindowsVideoMedia errors in some computers

We encountered some problems that some Windows computers were having trouble playing H264 videos.

Solution: Re-processed the videos with specific color space information.

Challenge 2: Problems with some AMD graphic cards

Some AMD computers were facing some trouble playing the game.

Solution: Switched the project's rendering pipeline to Vulkan.

Challenge 3: Emergency changes right before release date

Feedback on the UI required urgent changes under a tight schedule.

Solution: Discussed with our partner to reduce unnecessary workload. Coordinatied the team to deliever the changes on time.

What I Learned

  • Usage of FFmpeg
  • Unity videoplayer is relying on WMF
  • More traditional way to install a game by utilizing Innosetup
  • Personnel changes can postpone the development timeline significantly
  • Film industry working culture
